# Braydon Hammond — Computer Science ePortfolio

This repository hosts my CS 499 Computer Science Capstone ePortfolio, built for Southern New Hampshire University's Bachelor of Science in Computer Science program. The site is published via GitHub Pages at:

**https://braydonhammondsnhu.github.io**

## About

This ePortfolio documents the culmination of my Computer Science coursework, presented as a single capstone case study: the progressive enhancement of one Android application across the three foundational areas of the discipline.

- Software Design & Engineering
- Algorithms & Data Structures
- Databases

Rather than enhancing three unrelated class projects, I chose to take one application — an Android Inventory Manager originally built in CS 360: Mobile Architecture and Programming — through a complete, professional-grade transformation across all three categories.

## Site Structure

| File | Description |

## Course Outcomes

Each enhancement page documents the specific Course Outcomes it addresses. Across the capstone as a whole:

- **Outcome 1** (collaborative environments) — addressed primarily through the code review video
- **Outcome 2** (professional communication) — addressed through all written narratives and documentation
- **Outcome 3** (algorithmic design and trade-offs) — addressed in the Algorithms & Data Structures enhancement
- **Outcome 4** (well-founded techniques and tools) — addressed across all three enhancements
- **Outcome 5** (security mindset) — addressed primarily in the Databases enhancement

## Technology

This site is built with plain HTML and CSS — no build process or framework required. It is served directly via GitHub Pages from the repository root.
