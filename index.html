Overview

This ePortfolio documents the culmination of my Computer Science coursework at Southern New Hampshire University, presented as a single capstone case study: the progressive enhancement of one Android application across the three foundational areas of the discipline — software engineering and design, algorithms and data structures, and databases.

Begin with the Professional Self-Assessment for a full introduction to my background, skills, and how the artifacts below fit together. Each enhancement page includes the original and enhanced source code, a recorded code review, and a written narrative reflecting on the process.

The Artifact

All three enhancements below are applied to a single artifact: an Android Inventory Manager, originally built in Java for CS 360 (Mobile Architecture and Programming). The application allows a user to log in, manage a personal inventory of items, and receive low-stock alerts. Rather than enhancing three unrelated class projects, I chose to take this one application through a complete, professional-grade transformation — the same way a real piece of software might evolve from a prototype into a production-ready product.
